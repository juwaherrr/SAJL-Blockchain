### Metrics MicroPack

This folder contains performance and deployment metrics for the SAJL blockchain project.  
Each file provides a record of measurements collected during testing.

#### Files
| File Name | Description |
|------------|-------------|
| `gas-snapshot.txt` | Shows average gas usage and cost from smart contract runs. |
| `deploy.json` | Includes deployment information such as contract addresses and network data. |
| `coverage-summary.txt` | Summarizes test coverage results for the smart contracts. |
| `latency.csv` | Lists transaction latency and execution times from different tests. |

#### Purpose
These files are used to track system performance and support analysis during development and sprint reviews.
